Drug-Target Affinity Prediction Serve

Core Features

Affinity Prediction

Given a protein target and a candidate small molecule, our model performs structure-aware prediction of pIC50-based drug–target affinity​ by jointly encoding pocket-level protein features and 3D ligand conformations.

基于pIC50指标,联合编码口袋级蛋白质特征与3D配体构象,实现结构感知的药物-靶点亲和力精准预测，支持多格式输入与批量处理。

Input : Drug(Ligand SMILES/SDF)+ Target(UniProt/PDB/CIF)

Output : pIC50 Affinity value (2 decimal places)

Mode : Demo/Advanced

Molecular generation & lead optimization

Given a target and a known binder (lead compound), our model generates analogues with potentially improved affinity and better drug-likeness.

以已知靶点-结合剂为参考，生成具有更高亲和力、更优成药性的药物类似物，多维度指标评估分子特性，支持自定义筛选与数量设置。

Input : Target(UniProt/PDB/CIF)+ Reference drug(Ligand SMILES/SDF)

Output : generated SMILES、2D ligand conformations、predicted pIC50、ΔpIC50、LogP、SAS、QED

Mode : Demo/Advanced
